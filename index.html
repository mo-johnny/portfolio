Some of my personal experiments and projects working with various AI tools to create functional prototypes

Teacher Newsletter Agent

Created an agent using Cursor to generate a weekly newsletter for a classroom teacher based on a photo of the weekly lesson planner. The goal is to see how close to a teachers tone LLMs can get with minimum input. This could save valuable time on a task that most classroom teachers do every Friday or, even worse, on their weekends.

A playful web app that turns math fact practice into fast games designed to address the boredom and disengagement many learners feel with traditional math games. I used Google Stich for various UI options and Claude and Cursor to turn the design into a playable web app.

Softball Strategy

A web app that helps youth softball players learn and practice where to throw the ball on defense so they can confidently call out the play during the pressure of a real game. I started with Google Stitch and cleaned up the UI in Adobe Illustrator. I then used Cursor to make it playable.
